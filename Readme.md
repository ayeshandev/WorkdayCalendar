# Workday Calender

## Description

This project provides a simple way to calculate workdays, taking into account weekends and holidays. It allows you to add or subtract a specified number of workdays from a given date.

## Approach

1) Initilal project setup with domain models and interfaces.
2) Implement holiday management to add different types of holidays.
3) Implement a basic logic for calculating workdays, considering only forward calculations.
4) Extend the logic to handle backward calculations.
